Web Search and Mining - Google’s Intern Program

Oct. 2013 - Feb. 2014

During the 6-month internship in Google Beijing, I focused on finding fixed-contents(like menus, titles, ads) on the web pages and determine whether they are informative parts by designing simiarlity function for every node in HTML and designing the detect algorthim. I did lots of experiments and managed to raise the recall of marginal part by about 2% with little increment of recall of the essential part.


Personality Analysis on Social Network - Student Research Program

Jul. 2013 - Dec. 2013

Our purpose is to build model to explore user’s personality through his social media use. After collecting data and analysis (using the tool of WEKA to machine learning and regression fitting), we built an app on renren.com(a social network like facebook in China) that can predict user’s personality by analyzing his status, blogs, photos, shares, and comments on the renren.com.
The analysis of photos and voices, most of the UI design and part of the cralwing work are done by me.
The project got Third Price at Tsinghua Challenge Cup. You can see the project here. P.S. Unfortunately, the server is broken, but you can run the server by running "python manage.py runserver" and then go to "http://127.0.0.1:8000" to see the work.
